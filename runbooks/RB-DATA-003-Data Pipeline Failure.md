# RB-DATA-003 - Data Pipeline Failure

## Overview

This runbook defines standardised procedures for diagnosing and resolving
failures in data ingestion, preprocessing, embedding generation, and
vector database synchronisation.

It supports data integrity and system reliability.

---

## Purpose

This runbook enables teams to:

- Restore disrupted data pipelines
- Maintain data freshness and accuracy
- Prevent downstream model degradation
- Ensure compliance with data governance standards

---

## Applicable Systems

This runbook applies to:

- ETL and ELT pipelines
- Embedding generation workflows
- Vector database ingestion processes
- Batch and streaming pipelines

---

## When to Use

Use this runbook when:

- Data ingestion jobs fail
- Embedding generation stalls
- Knowledge bases become outdated
- Synchronisation gaps are detected
