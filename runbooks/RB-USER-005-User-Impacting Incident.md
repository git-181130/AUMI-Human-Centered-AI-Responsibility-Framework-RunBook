# RB-USER-005 - User-Impacting Incident

## Overview

This runbook defines standardised response procedures for incidents that negatively affect users, including harmful outputs, broken workflows, or poor interaction quality.

It supports user protection and brand trust.

---

## Purpose

This runbook enables teams to:

- Rapidly mitigate harmful user experiences
- Restore functional workflows
- Strengthen safety and tone controls
- Maintain accountability and compliance

---

## Applicable Systems

This runbook applies to:

- Customer-facing chatbots
- Workflow automation agents
- Support assistants
- Interactive AI services

---

## When to Use

Use this runbook when:

- Users report harmful or offensive outputs
- Task flows break or loop
- Safety guardrails fail
- Support escalations increase
